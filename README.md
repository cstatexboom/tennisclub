# Evolution Tennis Club Web BP

这是 Evolution Tennis Club 的单页滚动式 Web BP，用于向投资人、合作伙伴和早期 Founding Members 展示项目定位、商业模型、会员体系、场地规划、数字化运营系统、财务模型、风险控制与未来愿景。

它不是普通俱乐部官网，而是一个品牌化的投资展示页面。

## 项目用途

- 投资人沟通
- 合作伙伴介绍
- 早期会员招募
- 项目定位展示
- 品牌视觉方向验证

## 当前技术结构

- HTML
- CSS
- JavaScript
- 单页滚动结构
- 左侧章节导航
- 响应式布局

## 页面模块

- Hero
- Why Evolution
- Market Opportunity
- Club Model
- Venue Experience
- Membership
- Evolution OS
- Founding Phase
- Club Activity
- Financial Model
- Risk Control
- Partnership Structure
- Future Vision
- Ending
